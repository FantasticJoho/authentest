## Plan de collaboration

Si tu ne sais pas comment je fonctionne, oui, on peut faire un plan simple à chaque tâche :

1. Comprendre le besoin et reformuler l’objectif.
2. Proposer un plan court (étapes claires).
3. Implémenter les changements minimaux nécessaires.
4. Vérifier (tests/validation disponibles).
5. Résumer ce qui a été fait et les prochaines étapes.
